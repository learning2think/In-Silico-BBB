    bbb.data.substances

    Модуль substances — реестр модельных веществ для симуляции ГЭБ.

    Каждое вещество описывается датаклассом Substance с биофизическими параметрами транспортной модели и метаданными для UI.

    Реестр (7 веществ)

    Параметры адаптированы из литературы для безразмерной двухкомпартментной модели (k_pass [1/ч], Vmax [мкМ/ч], Km [мкМ]):

    Диазепам — высокая липофильность (logP≈2.8), слабый субстрат P-gp. Ref: Hammarlund-Udenaes et al., Drug Metab. Dispos., 2008. Верапамил — умеренная липофильность (logP≈3.8), выраженный P-gp. Ref: Choo et al., Br. J. Pharmacol., 2006. Декстран — гидрофильный полисахарид (>10 кДа), маркер барьера. Ref: Hawkins & Davis, Pharmacol. Rev., 2005. Кофеин — малая молекула (194 Да), не субстрат P-gp. Ref: Fredholm et al., Pharmacol. Rev., 1999. Морфин — гидрофильный (logP≈0.9), выраженный субстрат P-gp. Ref: Groenendaal et al., Drug Metab. Dispos., 2007. Донепезил — высокая липофильность (logP≈4.0), слабый субстрат P-gp. Ref: Tiseo et al., Clin. Pharmacokinet., 1998. Леветирацетам — гидрофильный (logP≈-0.6), не субстрат P-gp. Ref: Potschka et al., Epilepsia, 2004.

    @dataclass
class Substance:
    """
    Параметры вещества для двухкомпартментной модели ГЭБ.

    Атрибуты
    --------
    name : str
        Название вещества (отображается в интерфейсе).
    k_pass : float
        Коэффициент пассивной проницаемости [1/ч].
        Характеризует скорость пассивной диффузии через липидный бислой.
    vmax : float
        Максимальная скорость P-gp эффлюкса [мкМ/ч].
        0.0 означает, что вещество не является субстратом P-gp.
    km : float
        Константа Михаэлиса для P-gp [мкМ].
        Концентрация, при которой скорость эффлюкса равна Vmax / 2.
    color : str
        Цвет линии на графике (HEX-код).
    description : str
        Краткое описание вещества для интерфейса.
    reference : str
        Источник литературных значений параметров.
    """

    name: str
    k_pass: float  # [1/ч]
    vmax: float  # [мкМ/ч]
    km: float  # [мкМ]
    color: str = "#1f77b4"
    description: str = ""
    reference: str = ""
    fu_plasma: float = 1.0  # свободная фракция в плазме (0–1)
    fu_brain: float = 1.0  # свободная фракция в ткани мозга (0–1)

    def get_substance(name: str) -> Substance:
    """
    Получить вещество из реестра по названию.

    Параметры
    ---------
    name : str
        Название вещества (должно совпадать с ключом в SUBSTANCES).

    Возвращает
    ----------
    Substance
        Объект с параметрами вещества.

    Исключения
    ----------
    KeyError
        Если вещество с данным названием не найдено в реестре.
    """
    if name not in SUBSTANCES:
        available = ", ".join(SUBSTANCES.keys())
        raise KeyError(
            f"Вещество '{name}' не найдено в реестре. "
            f"Доступные вещества: {available}."
        )
    return SUBSTANCES[name]
