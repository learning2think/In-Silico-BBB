## Описание проекта

Гематоэнцефалический барьер — высокоселективный физиологический барьер,
отделяющий кровоток от ткани мозга. Около **98% малых молекул** не проникают
через ГЭБ в терапевтических концентрациях, что делает доклиническую оценку
проницаемости критически важным этапом разработки препаратов для лечения
заболеваний ЦНС.

**In Silico BBB** предлагает механистический вычислительный подход:
двухкомпартментная кинетическая модель учитывает как пассивную диффузию
(основной механизм для большинства малых молекул), так и активный эффлюкс
через P-гликопротеин — ключевой транспортёр, ограничивающий накопление
многих лекарств в мозге.

**Целевая аудитория:** исследователи в области фармакологии, медицинской химии
и нейронаук, работающие на этапе доклинической разработки и нуждающиеся
в быстром предварительном ранжировании соединений по проницаемости ГЭБ.

## Математическая модель

### Компартменты

| Компартмент | Биологический аналог | Объём (усл. ед.) |
|---|---|---|
| Апикальный | Кровь / донорная лунка | V_blood = 1.0 |
| Базолатеральный | Ткань мозга / акцепторная лунка | V_brain = 0.3 |

### Транспортные механизмы

**Пассивная диффузия** (закон Фика):

```
J_diff = k_pass × (C_blood − C_brain)    [мкМ/ч]
```

**Активный эффлюкс P-gp** (кинетика Михаэлиса–Ментен):

```
J_P-gp = Vmax × C_brain / (Km + C_brain)    [мкМ/ч]
```

### Система ОДУ

```
dC_blood/dt = −J_diff + (V_brain/V_blood) × J_P-gp
dC_brain/dt = (V_blood/V_brain) × J_diff − J_P-gp
```

Коэффициенты объёмов обеспечивают закон сохранения вещества.

### Выходные показатели

| Показатель | Описание |
|---|---|
| **C_brain(t)** | Кинетика концентрации в мозге [мкМ] |
| **Cmax** | Максимальная концентрация в мозге [мкМ] |
| **AUC** | Площадь под кривой в мозге [мкМ·ч] |
| **Kp,uu** | Отношение несвязанных концентраций мозг/кровь в равновесии |

### Численный метод

Используется **LSODA** (`scipy.integrate.solve_ivp`) — метод с автоматическим
переключением между явными (Adams) и неявными (BDF) схемами. Выбор обусловлен
потенциальной жёсткостью системы при малых значениях Km и высоких концентрациях.
Абсолютная точность: 1e-10, относительная: 1e-8.

## Установка и запуск

### Требования

- Python 3.11+
- pip

### Шаги

```bash
# 1. Клонировать репозиторий
git clone <url>
cd "In Silico BBB"

# 2. Создать виртуальное окружение
python -m venv .venv

# Windows:
.venv\Scripts\activate
# macOS / Linux:
source .venv/bin/activate

# 3. Установить зависимости
pip install -r requirements.txt

# 4. Запустить приложение
streamlit run app.py
```

После запуска браузер автоматически откроет страницу по адресу
`http://localhost:8501`.

### Добавление нового вещества

В файле `bbb/data/substances.py` добавьте запись в словарь `SUBSTANCES`:

```python
from bbb.data.substances import SUBSTANCES, Substance

SUBSTANCES["МоёВещество"] = Substance(
    name="МоёВещество",
    k_pass=0.4,
    vmax=1.2,
    km=15.0,
    color="#9C27B0",
    description="Описание вещества.",
    reference="Автор, журнал, год.",
)
```

## Научная новизна и планы развития

### Актуальность и научная новизна

Существующие in silico подходы к оценке проницаемости ГЭБ преимущественно
основаны на QSAR-моделях, предсказывающих бинарный исход (проникает / не
проникает). **In Silico BBB** предлагает механистическую кинетическую модель,
позволяющую:

- получать **временну́ю динамику** концентраций (а не только равновесное значение);
- разделять вклады **пассивного** и **активного** транспорта;
- рассчитывать клинически значимые показатели: **Kp,uu**, **Cmax**, **AUC**;
- интерактивно исследовать чувствительность к параметрам модели.

Подход дополняет, а не заменяет эксперимент: он позволяет сократить число
неинформативных опытов и сфокусировать ресурсы на наиболее перспективных
соединениях.

### Планы развития (этапы УМНИК)

**Этап 1 (год 1):**
- Интеграция QSAR-блока: предсказание k_pass из структуры молекулы
  (дескрипторы RDKit, модели случайного леса).
- Добавление транспортёров BCRP и MRP4 с независимыми кинетическими параметрами.
- Расширение реестра до 50+ веществ с верифицированными параметрами.

**Этап 2 (год 2):**
- Трёхкомпартментная модель клеточного монослоя для сравнения с данными
  Transwell и PAMPA-BBB.
- REST API для пакетного скрининга виртуальных библиотек соединений.
- Валидация на опубликованных PET-данных (мышь, крыса, человек).

**Долгосрочная перспектива:**
- Интеграция с платформами молекулярного докинга (AutoDock, Glide) для
  предсказания аффинности к транспортёрам.
- Конвертация в персонализированную модель на основе данных пациента
  (физиологически обоснованная фармакокинетика, PBPK).
