## Описание проекта

Гематоэнцефалический барьер — высокоселективный физиологический барьер,
отделяющий кровоток от ткани мозга. Около **98% малых молекул** не проникают
через ГЭБ в терапевтических концентрациях, что делает доклиническую оценку
проницаемости критически важным этапом разработки ЦНС-препаратов.

**In Silico BBB** предлагает механистический вычислительный подход:
двухкомпартментная кинетическая модель учитывает как пассивную диффузию
(закон Фика), так и активный эффлюкс через P-гликопротеин (Михаэлис–Ментен)
— ключевой транспортёр, ограничивающий накопление многих лекарств в мозге.

**Целевая аудитория:** исследователи в области фармакологии, медицинской химии
и нейронаук, работающие на этапе доклинической разработки.

## Математическая модель

### Компартменты

| Компартмент | Биологический аналог | Объём (усл. ед.) |
|---|---|---|
| Апикальный | Кровь / донорная лунка | V_blood = 1.0 |
| Базолатеральный | Ткань мозга / акцепторная лунка | V_brain = 0.3 |

### Транспортные механизмы

**Пассивная диффузия** (закон Фика):

```
J_diff = k_pass × (C_blood − C_brain)    [мкМ/ч]
```

**Активный эффлюкс P-gp** (кинетика Михаэлиса–Ментен):

```
J_P-gp = Vmax × C_brain / (Km + C_brain)    [мкМ/ч]
```

### Система ОДУ

```
dC_blood/dt = −J_diff + (V_brain/V_blood) × J_P-gp
dC_brain/dt = (V_blood/V_brain) × J_diff − J_P-gp
```

Объёмные коэффициенты обеспечивают **закон сохранения вещества**:
`d/dt(C_blood × V_blood + C_brain × V_brain) = 0`.

### Выходные показатели

| Показатель | Описание |
|---|---|
| **C_brain(t)** | Кинетика концентрации в мозге [мкМ] |
| **Cmax** | Максимальная концентрация в мозге [мкМ] |
| **AUC** | Площадь под кривой (метод трапеций) [мкМ·ч] |
| **Kp,uu** | Среднее C_brain/C_blood за последние 5 % симуляции |

### Численный метод

**LSODA** (`scipy.integrate.solve_ivp`) — автоматическое переключение между
Adams (нежёсткие участки) и BDF (жёсткие участки).
Точность: `atol=1e-10`, `rtol=1e-8`.

## Установка и запуск

### Требования

- Python 3.11+
- pip

### Шаги

```bash
# 1. Клонировать репозиторий
git clone <url>
cd "In-Silico-BBB"

# 2. Создать виртуальное окружение
python -m venv .venv

# Windows:
.venv\Scripts\activate
# macOS / Linux:
source .venv/bin/activate

# 3. Установить зависимости
pip install -r requirements.txt

# 4. Запустить приложение
streamlit run app.py
```

После запуска браузер автоматически откроет `http://localhost:8501`.

## Тесты

Проект покрыт 17 автотестами, проверяющими физику модели и корректность кода:

```bash
pytest tests/ -v
```

| Тест | Что проверяет |
|---|---|
| `test_mass_conservation_no_pgp` | Закон сохранения вещества без P-gp |
| `test_mass_conservation_with_pgp` | Закон сохранения вещества с P-gp |
| `test_passive_diffusion_equilibrium` | Диффузионное равновесие C_blood ≈ C_brain |
| `test_pgp_reduces_brain_concentration` | P-gp снижает Cmax и AUC в мозге |
| `test_simulation_success_for_all_substances` | Успешный запуск для всех 7 веществ |
| `test_ode_rhs_mass_conservation` | d/dt(total) = 0 в правых частях ОДУ |
| `test_auc_and_cmax_positive` | AUC > 0 и Cmax > 0 при ненулевой дозе |
| `test_zero_initial_concentration` | Нулевые начальные условия → нулевые концентрации |

## Планы развития

### Этап 1 (год 1 УМНИК)

- Интеграция QSAR-блока: предсказание k_pass из структуры молекулы
  (дескрипторы RDKit, модели случайного леса).
- Добавление транспортёров BCRP и MRP4 с независимыми параметрами.
- Расширение реестра до 50+ верифицированных веществ.

### Этап 2 (год 2 УМНИК)

- Трёхкомпартментная модель (донор / эндотелий / акцептор) для сравнения
  с данными Transwell и PAMPA-BBB.
- REST API для пакетного скрининга виртуальных библиотек.
- Валидация на опубликованных PET-данных (мышь, крыса, человек).

### Долгосрочная перспектива

- Интеграция с платформами молекулярного докинга (AutoDock, Glide).
- Персонализированная PBPK-модель на основе данных пациента.
