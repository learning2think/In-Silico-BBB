# Parameters

Все параметры задаются через боковую панель Streamlit и передаются в `SimulationParams`.

## Параметры транспортной модели

| Параметр | Символ | Единицы | По умолчанию | Диапазон | Описание |
|---|---|---|---|---|---|
| `k_pass` | k_pass | 1/ч | 0.5 | 0.001 – 5.0 | Коэффициент пассивной проницаемости. Характеризует скорость диффузии через липидный бислой по закону Фика. |
| `vmax` | V_max | мкМ/ч | 1.0 | 0.0 – 10.0 | Максимальная скорость P-gp эффлюкса. 0 означает отсутствие субстратности к P-gp. |
| `km` | K_m | мкМ | 10.0 | 0.1 – 500.0 | Константа Михаэлиса для P-gp. Концентрация, при которой скорость эффлюкса равна V_max / 2. |
| `k_elim` | k_elim | 1/ч | 0.0 | 0.0 – 2.0 | Константа системного клиренса из крови (k_elim = CL / V_blood). 0 = in vitro (закрытая система). |

## Начальные условия

| Параметр | Символ | Единицы | По умолчанию | Описание |
|---|---|---|---|---|
| `c0_blood` | C₀_blood | мкМ | 1.0 | Начальная концентрация в кровяном компартменте. |
| `c0_brain` | C₀_brain | мкМ | 0.0 | Начальная концентрация в мозговом компартменте. |

## Параметры компартментов

| Параметр | Символ | Единицы | По умолчанию | Описание |
|---|---|---|---|---|
| `v_blood` | V_blood | усл. ед. | 1.0 | Относительный объём кровяного (апикального) компартмента. |
| `v_brain` | V_brain | усл. ед. | 0.3 | Относительный объём мозгового (базолатерального) компартмента. |

## Параметры белкового связывания

| Параметр | Символ | Диапазон | Описание |
|---|---|---|---|
| `fu_plasma` | fu_plasma | 0.0 – 1.0 | Свободная (несвязанная) фракция вещества в плазме крови. |
| `fu_brain` | fu_brain | 0.0 – 1.0 | Свободная фракция в ткани мозга. |

Истинный несвязанный коэффициент распределения:

```
Kp,uu = (C_brain · fu_brain) / (C_blood · fu_plasma)
```

Рассчитывается как среднее за последние 5 % временного ряда симуляции.

## Параметры симуляции

| Параметр | По умолчанию | Описание |
|---|---|---|
| `t_end` | 24.0 ч | Время окончания симуляции. |
| `n_points` | 500 | Количество точек в выходном временном ряду. |

## Математическая модель

Система ОДУ:

```
dC_blood/dt = −k_pass·(C_blood − C_brain) + (V_brain/V_blood)·J_pgp − k_elim·C_blood
dC_brain/dt =  (V_blood/V_brain)·k_pass·(C_blood − C_brain) − J_pgp

где J_pgp = Vmax · C_brain / (Km + C_brain)
```

Численное решение: LSODA (`scipy.integrate.solve_ivp`), `rtol=1e-8`, `atol=1e-10`.
