# Interpreting Charts

## График концентраций (Results tab)

График отображает динамику концентрации вещества в обоих компартментах:

```
Ось X — Время (ч)
Ось Y — Концентрация (мкМ)

──── Красная кривая  →  Кровь (апикальный компартмент)
──── Синяя кривая    →  Мозг (базолатеральный компартмент)
░░░░ Синяя заливка   →  AUC в мозге (площадь под кривой)
  ↑  Аннотация       →  Cmax и t(Cmax) мозгового компартмента
```

**Типичные паттерны:**

| Форма кривой мозга | Интерпретация |
|---|---|
| Быстрый подъём, высокое плато | Высокая k_pass, слабый P-gp — хорошее проникновение |
| Медленный подъём, низкое плато | Слабая диффузия или выраженный P-gp эффлюкс |
| Кривая мозга почти нулевая | Декстран-подобный паттерн — барьер интактен |
| Кривая крови убывает, мозг растёт | Перераспределение вещества кровь → мозг |

!!! info "Метрики под графиком"
    - **Cmax** — пиковая концентрация в мозге [мкМ]
    - **t(Cmax)** — время достижения пика [ч]
    - **AUC** — интегральная экспозиция (трапеции) [мкМ·ч]
    - **Kp,uu** — несвязанный коэффициент распределения (учитывает fu_plasma, fu_brain)

## График анализа чувствительности (Sensitivity tab)

Веерный график: каждая кривая соответствует одному значению варьируемого параметра.

```
Цветовая шкала — Plasma (от фиолетового к жёлтому)
Нижние кривые  — малые значения параметра
Верхние кривые — большие значения параметра
```

**Параметры для анализа:**

| Параметр | Типичный результат при увеличении |
|---|---|
| **k_pass ↑** | Кривые смещаются вверх — больше вещества проникает в мозг |
| **Vmax ↑** | Кривые смещаются вниз — P-gp сильнее откачивает вещество из мозга |
| **Km ↑** | Кривые смещаются вверх — P-gp насыщается при меньших концентрациях |

!!! tip "Выбор компартмента"
    Переключатель **Brain / Blood** меняет отображаемый компартмент.
    Brain — целевой для ЦНС-препаратов.
    Blood — важен для оценки системной экспозиции.
